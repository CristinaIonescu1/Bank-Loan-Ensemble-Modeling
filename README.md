# Bank-Loan-Ensemble-Modeling
Predicting bank loan acceptance using Ensemble Learning techniques. A comparative study of Decision Trees, Random Forest, and Gradient Boosting, focusing on model performance, ROC curves, and learning stability.  
This project addresses the financial challenge of predicting personal loan acceptance by leveraging Ensemble Learning techniques. Using a dataset of customer profiles, I performed comprehensive data exploration and preprocessing followed by a comparative analysis of three powerful algorithms: Random Forest, Gradient Boosting, and Decision Trees. The Random Forest model emerged as the top performer, achieving a near-perfect accuracy score of 98.8%. The project demonstrates a full machine learning pipeline, including data splitting, model training, and performance benchmarking using Seaborn visualizations to identify the most effective predictive strategy for banking institutions.
